# Minecraft 마검 명령어

## Sword

### 정상 풀 인첸트

```c
/give @p netherite_sword{Enchantments:[{id:"unbreaking",lvl:3},{id:"sharpness",lvl:5},{id:"fire_aspect",lvl:2},{id:"looting",lvl:3},{id:"mending",lvl:1}]}
```

### Favorite Cheated Sword (unbreaking X)

```c
/give @p netherite_sword{Enchantments:[{id:"sharpness",lvl:5},{id:"fire_aspect",lvl:2},{id:"looting",lvl:3},{id:"mending",lvl:1}]}
```

### Favorite Cheated Sword (unbreaking O)

```c
/give @p netherite_sword{Enchantments:[{id:"sharpness",lvl:5},{id:"fire_aspect",lvl:2},{id:"looting",lvl:3},{id:"mending",lvl:1},{id:"unbreaking",lvl:255}]}
```

### 마검 그 자체

```c
/give @p netherite_sword{Enchantments:[{id:"unbreaking",lvl:255},{id:"sharpness",lvl:255},{id:"fire_aspect",lvl:255},{id:"looting",lvl:255},{id:"mending",lvl:255}]}
```
